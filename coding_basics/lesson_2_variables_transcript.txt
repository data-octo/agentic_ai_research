Lesson 2: Variables and Data Types

Welcome to Lesson 2! In this lesson, we will learn about variables and data types in Python.

1. **What is a Variable?**
   - A variable is like a container that stores data.
   - Example: Use a variable to store the balance of a bank account.

2. **Data Types in Python**
   - Python has different data types, such as:
     - Numbers (e.g., 1000, 0.05)
     - Strings (e.g., "Account Number")
     - Lists (e.g., [100, -50, 200])
   - Example: Store account details using variables with different data types.

By the end of this lesson, you will know how to use variables and data types to represent real-world data in Python. Let's dive in!
