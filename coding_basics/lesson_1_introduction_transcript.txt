Lesson 1: Introduction to Python and Banking

Welcome to Lesson 1! In this lesson, we will explore the basics of Python programming and how it is used in the banking industry.

1. **What is Python?**
   - Python is a beginner-friendly programming language.
   - It is widely used in industries like banking, healthcare, and technology.
   - Example: Write your first Python program to print "Hello, World!"

2. **How is Python used in Banking?**
   - Python helps manage customer accounts, calculate interest, and detect fraud.
   - It simplifies complex tasks and improves efficiency.

By the end of this lesson, you will understand why Python is a great choice for learning programming and its relevance to banking. Let's get started!
