Lesson 3: Conditional Statements and Loops

Welcome to Lesson 3! In this lesson, we will explore how to make decisions and repeat tasks in Python.

1. **Conditional Statements**
   - Use `if-else` statements to make decisions in your code.
   - Example: Check if a withdrawal amount is greater than the account balance.

2. **Loops**
   - Use loops to perform repetitive tasks.
   - Example: Iterate through a list of transactions to update the account balance.

By the end of this lesson, you will be able to write Python programs that make decisions and handle repetitive tasks efficiently. Let's get started!
